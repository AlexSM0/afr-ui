# AFR UI

A modern and reusable **component library built and maintained by me**.

`afr-ui` is my personal UI library, created to centralize and standardize components across my projects.
It also serves as a **learning project**, where I practice and apply best practices in modern frontend development.

---

## ✨ About

This project is:

* a **reusable component library** for my applications
* a **learning playground** to improve my skills

Focused on:

* Reusability
* Consistency
* Scalability
* Clean and maintainable code

---

## 🚀 Purpose

* Build my own design system for future projects
* Avoid rewriting common UI components
* Learn and apply solid development practices

---

## 🛠 Tech Stack

* React
* TypeScript
* Tailwind CSS
* Vite / Next.js

---

## 💡 Example Usage

```tsx
import { Button } from "afr-ui";

export default function App() {
  return <Button>Click me</Button>;
}
```

A growing UI library that evolves alongside my skills and supports my future personal projects.
